# asaitec-fruit-shop
 Asaitec Java Technical Test
 
 
 #### JAR
 
 You can find JAR in:
 https://github.com/limeres/asaitec-fruit-shop/tree/master/files 
 
 For run app:
 
 ```
 java -jar shop-0.0.1-SNAPSHOT.jar
 ```
 
 This app runs in:
 
  ```
 localhost:8080
 ```
